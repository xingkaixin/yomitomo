发布日期：2026-05-11

## 新功能

- 桌面端阅读库支持通过 URL 导入网页文章，导入时可持久化正文图片，并补齐阅读器正文的基础排版。
- 阅读器新增选区复制与快捷键、批注动作和类型图标、右侧批注筛选。
- 消息发送快捷键支持在设置中配置，并在阅读器发送、取消、提示文案中保持一致。
- 用户批注支持异步推断批注类型和阅读意图，`@` 助手消息可拆分为多助手任务规划。
- 新增重点共读流程，支持章节卡片、手动分配助手、章节消息和基于文章分析的路由生成。

## 修复

- 修复用户批注意图和快捷键持久化、快捷键提示归位、批注卡片单击展开，以及 SubmitShortcutKeys 组件排版问题。
- 修复 Provider 编辑器在紧凑布局中 select 菜单被 dialog overlay 遮挡的问题。
- 修复助手阅读章节遗漏正文开头内容、受页脚浅层标题影响导致章节规划失真的问题。
- 修复自动批注标签展示过长和重点共读合入后用户批注评论恢复的问题。
- 修复 lint/format 任务按根级单任务运行的问题，改为 workspace package 级调度。

## 文档

- 更新开发代理文档中的 lint/format 流程，并清理 Chrome extension 相关说明、隐私文本、第三方声明和分发资料。

## 杂项

- 移除 Chrome extension 工作区、运行时、桌面配对桥接和上架资产，产品线收敛到桌面端应用。
- 新增 GitHub Actions CI，在 PR 和 main 推送中运行 lint、format check、test 和 build，并固定检查时区。
