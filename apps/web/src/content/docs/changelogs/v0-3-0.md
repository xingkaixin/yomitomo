发布日期：2026-05-15

## 新功能

- 桌面端新增本地 EPUB 导入与阅读，支持导入对话框、封面和章节元数据保存，并通过 Foliate 阅读器打开电子书。
- EPUB 阅读链路新增结构化书籍索引、paragraph-aware 文本锚点和 segment 级批注任务。
- AI 伴读新增剧透范围控制、阅读上下文打包、选区上下文、thread-first 回复上下文、descriptor 路由和 EPUB 阅读记忆。
- EPUB 共读新增同章 lexical related passages、评估矩阵和长片段切分。
- 阅读器新增助手阅读 dock、滚动边缘模糊、批注导航快捷键、长批注折叠和助手 dock 动画。
- 阅读器选区复制和添加批注快捷键支持在设置中自定义，并在网页阅读器和 EPUB 阅读器中复用。

## 性能

- 新增 EPUB 性能埋点，记录导入、索引、批注生成和阅读器关键步骤耗时。
- 减少 EPUB 高亮重算，降低翻页、滚动和批注变化时的重复计算成本。

## 修复

- 修复嵌入式阅读器高度不足、精读助手控件焦点不稳定，以及短文场景下自动批注过多的问题。
- 修复阅读库排序变化后分组不同步的问题。
- 修正桌面端第三方许可证声明和生产依赖清单。

## 工程

- 新增 Turbo `typecheck` 任务，统一调度各 workspace package 的 TypeScript 类型检查。
