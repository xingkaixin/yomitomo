发布日期：2026-06-16

## 新功能

- 桌面端新增语言切换、拆分后的应用文案资源、官网双语文档与本地化助手角色展示。
- 阅读器新增持久聊天、正文搜索、统一悬浮工具栏、重设计的文章 header、空笔记状态与自适应批注布局；支持堆叠批注卡片平滑切换与选区高亮/下划线统一强调。
- 网页文章支持双语翻译；删除或重新翻译时会级联清理关联翻译批注；助手回复语言与界面语言对齐。
- 批注与讨论深化：根助手回复路由、沉淀稿与审阅建议体验、沉淀过渡动效、「添加想法」墨迹书写仪式与助手书写动画。
- 设置与资料库重做：内容来源定制、来源感知弹窗、方向感过渡、助手用量统计、黄昏靛蓝主题、助手档案详情与设置项信息 tooltip。
- 更新说明弹窗、本地批注写入、媒体封面重做、手绘墨迹纸张/主题选择、品牌音标发音。
- 统一 Toast（含微信读书）、应用内音效与响度，以及高亮、复制、导入/删除、沉淀等操作反馈。
- 官网新增阅读器风格落地页与博客场景内容。

## 性能

- 阅读记忆、EPUB 索引与助手摘要复用更多预计算与 SQL 聚合；Markdown 渲染记忆化；导入生命周期与 source reader 工作区进一步解耦。

## 修复

- 设置、阅读器、资料库封面、批注侧栏分组与沉淀锚定、讨论与想法交互、统计计入助手贡献。
- SQLite native root 与运行时依赖打包、记忆回填重试、网页 HTML/Markdown 消毒与 renderer 沙箱。
- 官网落地页响应式与轮播横向滚动。

## 工程

- 大规模模块拆分、UI 迁移至 Base UI、IPC 校验与结构化错误、E2E 冒烟基线；macOS 签名与公证发布流水线。
