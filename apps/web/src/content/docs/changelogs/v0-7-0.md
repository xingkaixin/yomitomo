发布日期：2026-06-08

## 新功能

- 桌面端新增语言切换，官网文档补齐中英文双语入口，助手角色展示也会按语言提供本地化说明。
- 阅读器新增持久聊天会话，围绕当前文章、EPUB 或 PDF 的对话可以在阅读过程中保留。
- 阅读器新增正文搜索，支持在阅读界面内快速定位关键词。
- 阅读器工具栏统一为悬浮操作面板，并重新设计文章 header、空笔记状态和可用空间下的批注布局。
- 批注与讨论链路继续深化，支持把回复路由到根助手，并在沉淀流程中生成更可执行的审阅建议。
- 设置页信息架构重做，在统计中展示助手用量，并新增资料库内容来源定制、来源感知弹窗和方向感过渡动画。
- 桌面端更新流程新增版本更新说明弹窗，更新前可读取官网发布说明，更新后可读取本地打包文案。
- 新增本地批注写入、PDF/网页文章媒体封面重做、电子书导入封面反馈和黄昏靛蓝主题。

## 性能

- 阅读记忆、EPUB 索引和助手摘要查询复用更多预计算结果，减少重复扫描与聚合。
- 文章仓库批量操作、文章导入生命周期和图片内联链路改为更稳定的编排方式，降低导入时的阻塞和失败面。
- 助手运行时、模型价格、微信读书同步和批注记忆链路逐步迁移到 Effect 编排，错误边界更清晰。

## 修复

- 修复设置页与阅读器多处视觉问题，包括设置 surface、菜单布局、标题栏文本约束、阅读器标题栏和批注卡片细节。
- 修复阅读库列表卡片、网页封面工具栏、扁平封面进度条和 PDF 封面比例问题。
- 修复讨论头像距离、添加想法输入器、回复空想法打开位置和想法自动滚动范围问题。
- 修复 SQLite native root 隔离、文章摘要计数范围、数据恢复失败路径和 provider keyring 回归风险。
- 修复官网产品轮播导致 landing page 横向滚动的问题。

## 工程

- 拆分桌面端 main、renderer、source reader、settings、PDFium、阅读器 shell 和全局 CSS 边界，降低单文件复杂度。
- 拆分 AI assistant runtime、provider、context、reading memory、shared/core 领域模块，并补充导出覆盖测试。
- 补充数据管理、store-db 恢复、文章导入 worker、provider secrets、阅读器聊天和阅读器 shell 状态测试。
- 新增 native root 验证、PR 模板、输出目录忽略规则，并更新 pnpm 与依赖版本。
