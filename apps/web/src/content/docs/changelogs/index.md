## 0.12.0

发布日期：2026-07-14

- 助手执行模式可在快速回应与深入查证之间切换，统一作用于批注、追问和共读。
- 合集整理支持键盘拖放与操作播报，阅读库网格会随可用宽度自适应排布。
- 数据库恢复改为可回滚的原子流程，网页导入与 API Key 日志边界进一步收紧。
- 默认主题统一为暖纸与朱砂视觉；官网补充移动端产品预览、滚动动效和可访问性改进。

[查看 0.12.0 更新记录](/changelogs/v0-12-0/)

## 0.11.0

发布日期：2026-07-05

- 批注卡片新增扇形堆叠与洗牌交互，密集批注栏更容易浏览。
- 自动化测试期间不再发送真实遥测心跳。
- 桌面端 E2E 覆盖扩展到阅读库导入、阅读器导航、批注、进度、设置、AI、EPUB/PDF 导入和响应式布局。
- 拆分阅读器、PDFium、网页 source、样式和文章仓库边界，降低后续改动风险。

[查看 0.11.0 更新记录](/changelogs/v0-11-0/)

## 0.10.0

发布日期：2026-06-28

- 阅读库新增「文本」类型：可粘贴文本或上传 `.txt` / `.md`，带稿纸封面、类型筛选和合集支持。
- 选区新增可拖动端点手柄，导入与模型路由新增隐私边界提示。
- 电子书点击翻页热区与自适应跨页布局，PDF 默认缩放自适应窗口。
- 禁用双击选区，修复封面刷新后被压缩，加固桌面安全边界。

[查看 0.10.0 更新记录](/changelogs/v0-10-0/)
