## 0.9.0

发布日期：2026-06-24

- 阅读库改为全类型混排，支持合集、置顶、会话内类型筛选和更完整的空状态。
- 沉淀审阅改为结构化流式建议，并可在草稿中预览、确认或放弃变更。
- 微信读书可自动同步，应用可自动检查更新；本地存储、缓存和阅读器细节更稳定。

[查看 0.9.0 更新记录](/changelogs/v0-9-0/)

## 0.8.0

发布日期：2026-06-20

- 新增 AZW3/MOBI 电子书导入与阅读，并修复 Kindle 内容的批注定位。
- 大型资料库、电子书分页、PDF 打开和助手诊断视图更轻快。
- 网页导入、EPUB 解压、应用锁和数据库备份边界更安全；数据管理设置更清楚。

[查看 0.8.0 更新记录](/changelogs/v0-8-0/)

## 0.7.2

发布日期：2026-06-17

- 新增应用锁，可用 PIN 保护本地阅读空间，并在设置中管理。
- 改进网页文章段落翻译反馈、阅读器目录高亮/聚焦与目录开合动效。
- 官网下载与应用更新链路改走 Cloudflare 下载 Worker，并采集下载与更新指标。

[查看 0.7.2 更新记录](/changelogs/v0-7-2/)
