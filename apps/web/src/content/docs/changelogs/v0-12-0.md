发布日期：2026-07-14

## 新功能

- 模型路由新增助手执行模式滑杆，可在「快速回应」与使用工具的「深入查证」之间选择，并全局作用于阅读批注、追问和共读任务。(#618)
- 合集选择器重做拖放交互，新增键盘拖动手柄、本地化操作播报和按钮回退，让不同输入方式都能一致地整理合集。(#616)
- 阅读库网格现在根据实际内容宽度响应式排布，卡片与加载骨架从紧凑窗口到宽窗口共用同一套布局。(#634)
- 官网新增滚动驱动的内容揭示、本地化移动端产品预览、更清晰的证据链版式，以及面向产品页和场景文章的结构化数据。(#619, #632, #637, #638)

## 性能

- 精简官网内容揭示动效，减少重复的观察器和动画工作，同时保留渐进增强回退。(#622)

## 修复

- 网页文章导入请求和渲染链路绑定到已校验的 IP，拦截 DNS rebinding 与保留地址绕过，进一步收紧本地网络边界。(#623)
- AI 运行日志不再记录 API Key 片段，只保留供应商身份和凭据是否已配置。(#624)
- 数据库恢复改为原子流程：替换前先校验同目录临时副本，保留回滚快照，并在切换失败时重新打开原数据库。(#625)
- 文章摘要分别保存想法数和讨论数，使批注详情尚未加载时的阅读统计仍保持准确。(#626)
- 键盘快捷键现在会立即开关阅读器聊天面板；鼠标操作仍保留可中断的面板动效。(#630)
- 改进官网的减弱动效适配、弱化文字对比度、键盘跳转导航和图片布局稳定性。(#631, #636, #640)

## 工程

- 命令行类型检查升级到原生 TypeScript 7；Astro 官网暂留 TypeScript 6，等待其编译器 API 支持 TypeScript 7。(#615)
- 更新 pnpm 与 workspace 依赖，修复 Vite shared chunk 下的桌面资源路径解析，并把桌面 E2E 纳入文档化的本地 CI 门禁。(#617, #627)
- 收窄桌面 IPC registrar 依赖，更新批注数据流文档，明确 shell/阅读库样式归属，精简阅读库过渡，压缩应用 header，并用暖纸中性色与统一朱砂强调色收拢默认主题表面。(#628, #629, #633, #635, #639, #641)
