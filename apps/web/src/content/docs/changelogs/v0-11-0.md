发布日期：2026-07-05

## 新功能

- 批注卡片新增扇形堆叠与洗牌交互。密集批注栏现在更容易浏览，重叠的笔记也能被翻到前面查看。(#590)

## 修复

- 自动化运行期间不再发送桌面端遥测心跳，避免本地和 CI 的 E2E 测试写入真实心跳事件。(#591)

## 工程

- 新增桌面端 UI harness，隔离 E2E 测试数据，并补齐阅读库导入、阅读器导航、批注、进度、设置、AI、EPUB/PDF 导入和响应式布局等 E2E 覆盖。(#592–#605)
- 拆分桌面端 renderer App 外壳、网页 source helper、PDFium 文档逻辑、阅读器搜索导航和选区调整规则，形成可复用且有测试覆盖的边界。(#606, #607, #610, #611, #612)
- 拆分 reader conversation CSS，移除遗留样式覆盖，补充桌面样式归属文档，并拆分文章仓库领域模块。(#608, #609, #613)
