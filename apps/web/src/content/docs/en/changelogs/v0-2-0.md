Released: 2026-05-11

## New

- Desktop Library can import web articles by URL, persist body images, and render baseline article typography.
- Reader added selection copy, shortcuts, annotation actions and type icons, and sidebar filtering.
- Message send shortcuts can be configured in Settings and are reflected across reader send, cancel, and hint copy.
- User annotations can infer annotation type and reading intent asynchronously, and `@` assistant messages can split into multi-assistant task planning.
- Added focused co-reading with chapter cards, manual assistant assignment, chapter messages, and article-analysis-based route generation.

## Fixes

- Fixed user annotation intent and shortcut persistence, shortcut hint placement, annotation card click-to-expand, and SubmitShortcutKeys layout.
- Fixed provider editor select menus being hidden behind dialog overlays in compact layouts.
- Fixed assistant chapter reading missing opening text and shallow footer headings distorting chapter planning.
- Fixed overly long automatic annotation tags and restored user annotation comments after focused co-reading was merged.
- Fixed lint/format tasks to run per workspace package instead of as root-level single tasks.

## Docs

- Updated development agent docs for lint/format flow and cleaned up Chrome extension, privacy, third-party notice, and distribution materials.

## Misc

- Removed the Chrome extension workspace, runtime, desktop pairing bridge, and store assets to focus the product line on the desktop app.
- Added GitHub Actions CI for PR and main pushes, running lint, format check, tests, and build with a fixed timezone.
