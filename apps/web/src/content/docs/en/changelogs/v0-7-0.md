Released: 2026-06-16

## New

- Desktop language switching, split app locale resources, bilingual website docs, and localized assistant persona presentations.
- Persistent reader chat, in-reader search, unified floating toolbar, redesigned article header, empty-note polish, adaptive annotation layouts, smooth stacked card switching, and unified selection emphasis.
- Bilingual web-article translation with cascade cleanup on delete/retranslate, and response language aligned with the UI locale.
- Deeper annotation and discussion flows: root-assistant reply routing, distillation draft and review-proposal UX, transition motion, ink-writing add-thought ceremony, and assistant writing animations.
- Reworked Settings and Library IA, assistant usage in stats, dusk indigo theme, assistant profile dialogs, and setting descriptions in info tooltips.
- Release-notes dialogs, local annotation writes, redesigned media covers, hand-drawn ink paper/theme pickers, and phonetic brand pronunciation.
- Unified toasts (including WeRead), sound-effect controls, and feedback for highlights, copy, library actions, and distillation.
- Reader-style website landing page and blog scenario articles.

## Performance

- More reuse of reading-memory, EPUB index, and SQL-backed assistant summaries; memoized Markdown rendering; cleaner import lifecycle and source-reader workspace boundaries.

## Fixes

- Settings, Reader, and Library polish; annotation-rail grouping and distillation anchoring; discussion and thought interactions; assistant contributions in stats.
- SQLite native-root packaging, memory-backfill retries, DOMPurify sanitization, and renderer sandbox.
- Website landing responsive layout and carousel horizontal scroll.

## Engineering

- Large module splits, Base UI migration, validated structured IPC, E2E smoke baseline, and signed/notarized macOS release pipelines.
