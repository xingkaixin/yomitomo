Released: 2026-06-26

## Fixes

- Fixed Library search clear duration parsing in production builds. Minified CSS may emit motion durations as `1s` / `.4s`; Yomitomo now converts those second-based values to milliseconds, so the clear button dissolve no longer ends too early.
