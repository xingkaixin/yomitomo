description: Yomitomo 0.8.0 release notes.
---

Released: 2026-06-20

## Features

- The Library can now import AZW3 and MOBI ebooks alongside EPUB, keeping covers, authors, chapters, reading progress, and the original source file.
- Kindle-format annotation location now falls back to source text, making it easier to return from notes to the original AZW3/MOBI passage.
- Data Management in Settings is clearer, with one place for the data directory, log file, database file, log retention, log clearing, backup, and restore.
- The website adds a FAQ section and includes FAQ, publisher, and screenshot structured data so search and AI citation surfaces can understand the product more accurately.

## Performance

- Large local libraries feel lighter because the article catalog is read in pages.
- Ebook pagination is now incremental; PDF text indexing is deferred until needed, and PDF source files avoid extra buffer copies.
- Assistant diagnostics trace details load on demand, and reading-memory text fallback now has bounded scanning.
- Desktop release packages are smaller after slimming reading fonts and bundled assistant persona imagery.

## Fixes

- Web-article import blocks localhost, private network, and cloud metadata addresses by default, and limits oversized HTML responses to reduce accidental private-resource imports.
- EPUB import rejects oversized decompressed entries; database backup and restore preserve target paths more reliably; App Lock now covers guarded IPC calls.
- Fixed reader annotation layout after width changes, in-reader search responsiveness, library source counts, overflowing PDF header authors, update-check feedback, and import success dialog timing.
- AZW3/MOBI reading no longer shows an annotation selector that does not apply to those formats.
