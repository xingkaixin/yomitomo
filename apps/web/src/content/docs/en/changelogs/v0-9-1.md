Released: 2026-06-25

## Features

- Reading-library search (main list and collections) and reader search gained dedicated clear actions that dissolve in place, while closing search stays a separate action.
- The reader Ask panel and its button now share a bottom-right morph for smoother open and close. All of these transitions honor the system reduced-motion setting.

## Fixes

- Restoring the last-read PDF page is faster, and jumping with the progress slider no longer waits on a stale page restore.
- Article edits made in a sedimentation window now sync to the main window, so Library summaries no longer show stale content.
- Fixed annotation-rail scrolling, web-translation drag selection visibility, the discussion mention-chip caret, and reduced-motion behavior in the source reader.
- Kept the dock icon source consistent and cleared the reading memory left behind after annotations or comments are removed.
