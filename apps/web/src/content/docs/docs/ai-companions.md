## 专注伴读

专注伴读适合精读一个章节或一组段落：

1. 在阅读器中打开专注伴读面板。
2. 选择一位或多位阅读助手。
3. 选择要阅读的段落范围。
4. 设置批注密度。
5. 启动伴读。

伴读开始后，助手会按段落生成批注，并尽量把批注落回对应原文。

## 上下文范围

助手生成批注时会参考：

- 当前段落的完整文本
- 你已有的批注和讨论
- 前文阅读记忆
- 文章目录和整体结构

EPUB 电子书还可以设置剧透范围，例如仅当前选段、当前章节、已读内容或全书。

| --- | --- | --- |

| 许问渠 | 追问导师 | 把模糊困惑转成精确问题 |
| 陈砚书 | 洞见整理员 | 提取可迁移洞见 |
| 沈清源 | 概念翻译员 | 解释术语和概念脉络 |
| 顾行简 | 结构领航员 | 识别文章结构和段落功能 |

## 审阅助手

审阅助手偏向证据、逻辑、表达和行动可行性复核：

| 助手 | 角色 | 擅长 |
| --- | --- | --- |
| 梁证言 | 证据司书 | 核实事实和证据链 |
| 叶听澜 | 读者权益官 | 关注真实疑问 |
| 唐简 | 终审编辑 | 压缩冗余、提升清晰度 |
| 何明衡 | 逻辑复核官 | 检测推理漏洞 |
| 苏定白 | 风险审查员 | 标记过度概括和风险 |
| 夏归宁 | 行动校准师 | 校准后续行动是否可执行 |

在「助手」页面可以启用或禁用助手，并查看各助手的角色说明。
