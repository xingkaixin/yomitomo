统计页帮助你回看阅读活动，而不是只看到材料列表。它包含「本地阅读」和「微信读书」两个来源。

## 统计指标

统计页会展示：

- 今日活动
- 已记录天数
- 每周活跃天数
- 活跃峰值
- 导入文章数
- 批注条数
- 沉淀条数
- 讨论次数
- 助手参与次数（含助手在统计摘要中的贡献）

活动图表可以查看文章导入、划线、沉淀和讨论评论的趋势。设置改版后，助手用量也会在统计相关视图中更易查看。活动日历会显示近 70 天的阅读活跃程度。

## 微信读书统计

配置微信读书 API KEY 后，可以在统计页切换到「微信读书」。微信读书统计支持按周、月、年和累计查询。切换周期只会查看已保存的数据；需要更新某个周期时，点击「查询」或「重新查询」。

微信读书统计会保存查询结果，之后即使不重新请求接口，也能查看已经缓存过的周期。

## AI 功能没有反应

请检查：

1. 是否已经添加至少一个模型供应商。
2. API Key 是否正确。
3. 「测试连接」是否通过。
4. 是否已经为对应任务配置路由。

## 网页文章导入失败

常见原因包括：

- 链接不是有效的 `http://` 或 `https://` 地址。
- 链接指向本机、局域网或云元数据地址，默认会被安全策略拦截。
- 网页 HTML 响应超过 5MB。
- 原站有反爬保护。
- 网络连接超时。

Yomitomo 会尝试用内置浏览器渲染页面，但少数网站仍可能无法解析。

## 电子书导入失败

请确认文件是 `.epub`、`.azw3` 或 `.mobi` 格式，且文件大小不超过 80MB。受 DRM 保护或解压后内容过大的文件无法导入。

## PDF 导入失败

请确认文件是标准 `.pdf` 格式，且文件大小不超过 120MB。

## 数据存储在哪里

所有数据存储在本地应用数据目录，不会上传到 Yomitomo 服务器。API Key 使用系统 keyring 保存。

## 是否发送匿名数据

Yomitomo 默认每天最多发送一次匿名心跳，只包含应用版本、系统版本和架构，用于了解活跃版本分布，不采集阅读内容、书名、批注、文件路径或 AI 对话。你可以在「设置 > 通用」的「隐私」分组中关闭「发送匿名版本与系统指标」停止上报。

## 支持哪些系统

目前支持 macOS 和 Windows。
