统计页帮助你回看阅读活动，而不是只看到材料列表。

## 统计指标

统计页会展示：

- 今日活动
- 已记录天数
- 每周活跃天数
- 活跃峰值
- 导入文章数
- 批注条数
- 讨论次数
- 助手参与次数

活动图表可以查看文章导入、批注创建和讨论评论的趋势。活动日历会显示近 70 天的阅读活跃程度。

## AI 功能没有反应

请检查：

1. 是否已经添加至少一个模型供应商。
2. API Key 是否正确。
3. 「测试连接」是否通过。
4. 是否已经为对应任务配置路由。

## 网页文章导入失败

常见原因包括：

- 链接不是有效的 `http://` 或 `https://` 地址。
- 原站有反爬保护。
- 网络连接超时。

Yomitomo 会尝试用内置浏览器渲染页面，但少数网站仍可能无法解析。

## EPUB 导入失败

请确认文件是标准 `.epub` 格式，且文件大小不超过 80MB。

## 数据存储在哪里

所有数据存储在本地应用数据目录，不会上传到 Yomitomo 服务器。API Key 使用系统 keyring 保存。

## 支持哪些系统

目前支持 macOS 和 Windows。
