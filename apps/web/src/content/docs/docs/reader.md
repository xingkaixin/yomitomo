## 阅读界面

阅读器包含正文区域、划线高亮、划线连线和侧边栏。顶部悬浮工具栏提供目录、搜索、划线导航等常用操作，以及围绕当前材料追问助手的持久聊天。网页文章还支持双语翻译，译文写回正文时会在对应段落给出反馈，便于确认哪些段落已经应用译文。

- **目录**：点击「切换目录」展开章节目录，目录会高亮并聚焦当前章节，方便从当前位置查看前后结构。
- **搜索正文**：在当前材料中搜索关键词，定位到正文位置。
- **划线快捷选择**：在上一条和下一条划线之间快速跳转，不必在正文里来回滚动。

阅读器内可以调整排版：

- 网页文章和电子书：**字号** 和 **文章宽度**。
- PDF：用 **PDF 缩放** 调整页面大小，对应网页文章/电子书的字号和页面宽度。

主题和阅读器纸张在**首页**的主题入口里切换（亮色、暗色、黄昏靛蓝等主题与阅读器纸张），不在阅读器内。纸张颜色会影响网页文章、电子书和 PDF 的正文阅读区域；暗色主题下 PDF 会保留原始页面颜色。

## 创建划线

1. 在正文中选中一段文本。
2. 按默认快捷键 `A`，或使用选区操作按钮。
3. 写下想法、疑问或笔记。

划线会以你的个人颜色高亮显示，并绑定到原文位置。创建划线、复制选区等操作可配合音效反馈（可在「设置 > 通用」中开关音效与响度）。

## 讨论区

每条划线都有独立讨论区。点击划线卡片上的「进入讨论区」后，可以继续写想法、回复已有想法，也可以在评论中 `@助手名称` 让 AI 助手加入讨论。助手回复会基于文章上下文、选区和已有讨论生成。

讨论区支持固定重要想法、删除自己的想法，并在左右分栏和左对齐布局之间切换。

## 沉淀

当一条划线已经积累出可保留的判断、框架、问题或行动提醒时，可以打开沉淀窗口，把分散想法整理成一段沉淀稿，并借助审阅助手的结构化反馈逐条校验、预览和采纳。详见「[沉淀与审阅](/docs/sedimentation/)」。

## 快捷键

| 快捷键 | 操作               |
| ------ | ------------------ |
| `C`    | 复制选中文本       |
| `A`    | 为选中文本创建划线 |
| `Q`    | 问一下选中文本     |

这些快捷键可以在「设置 > 快捷键」中自定义。
