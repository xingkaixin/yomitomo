## 阅读界面

阅读器包含正文区域、批注高亮、批注连线和侧边栏。EPUB 电子书还会显示章节目录和章节级划线、沉淀统计。

阅读器内可以调整：

- 字体大小
- 页面宽度

应用左侧的主题按钮可以切换亮色、暗色和阅读器纸张。纸张颜色会影响网页文章、EPUB 和 PDF 的正文阅读区域；暗色主题下 PDF 会保留原始页面颜色。

## 创建批注

1. 在正文中选中一段文本。
2. 按默认快捷键 `A`，或使用选区操作按钮。
3. 写下想法、疑问或笔记。

批注会以你的个人颜色高亮显示，并绑定到原文位置。

## 批注类型

Yomitomo 会把批注归入这些类型，帮助之后筛选和回看：

| 类型 | 说明 |
| --- | --- |
| 要点 | 文章的核心论点或关键信息 |
| 假设 | 作者的前提假设或隐含条件 |
| 概念 | 需要理解的术语或概念 |
| 疑问 | 你的困惑或待探讨的问题 |
| 引用 | 值得记录的原文 |

## 阅读意图

创建批注时可以选择阅读意图，告诉助手你希望得到怎样的回应：

| 意图 | 说明 |
| --- | --- |
| 解释 | 解释这段内容 |
| 拆解 | 分解论证结构 |
| 质疑 | 挑战这个观点 |
| 追问 | 提出后续问题 |
| 关联 | 关联其他知识 |

## 讨论区

每条划线都有独立讨论区。点击划线卡片上的「进入讨论区」后，可以继续写想法、回复已有想法，也可以在评论中 `@助手名称` 让 AI 助手加入讨论。助手回复会基于文章上下文、选区和已有讨论生成。

讨论区支持固定重要想法、删除自己的想法，并在左右分栏和左对齐布局之间切换。

## 沉淀

当一条划线已经积累出可保留的判断、框架、问题或行动提醒时，可以打开沉淀窗口，把分散想法整理成一段沉淀稿。

沉淀窗口中可以：

- 编辑沉淀稿。
- 选择一个或多个审阅助手，让它们围绕沉淀稿给出反馈。
- 发布沉淀，让它替代原始划线显示在侧边栏。
- 取消发布，继续保留沉淀稿和审阅记录。

## 快捷键

| 快捷键 | 操作 |
| --- | --- |
| `C` | 复制选中文本 |
| `A` | 为选中文本创建批注 |

这些快捷键可以在「设置 > 快捷键」中自定义。
