## 阅读界面

阅读器包含正文区域、批注高亮、批注连线和侧边栏。EPUB 电子书还会显示章节目录和章节级批注统计。

阅读器内可以调整：

- 字体大小
- 页面宽度

## 创建批注

1. 在正文中选中一段文本。
2. 按默认快捷键 `A`，或使用选区操作按钮。
3. 写下想法、疑问或笔记。

批注会以你的个人颜色高亮显示，并绑定到原文位置。

## 批注类型

Yomitomo 会把批注归入这些类型，帮助之后筛选和回看：

| 类型 | 说明 |
| --- | --- |
| 要点 | 文章的核心论点或关键信息 |
| 假设 | 作者的前提假设或隐含条件 |
| 概念 | 需要理解的术语或概念 |
| 疑问 | 你的困惑或待探讨的问题 |
| 引用 | 值得记录的原文 |

## 阅读意图

创建批注时可以选择阅读意图，告诉助手你希望得到怎样的回应：

| 意图 | 说明 |
| --- | --- |
| 解释 | 解释这段内容 |
| 拆解 | 分解论证结构 |
| 质疑 | 挑战这个观点 |
| 追问 | 提出后续问题 |
| 关联 | 关联其他知识 |

## 讨论线程

每条批注下方都有讨论线程。你可以继续写评论，也可以在评论中 `@助手名称` 让 AI 助手加入讨论。助手回复会基于文章上下文、选区和已有讨论生成。

## 快捷键

| 快捷键 | 操作 |
| --- | --- |
| `C` | 复制选中文本 |
| `A` | 为选中文本创建批注 |

这些快捷键可以在「设置 > 快捷键」中自定义。
