阅读库是所有阅读材料的入口。网页文章、EPUB/AZW3/MOBI 电子书、PDF 文档、微信读书笔记和合集会分开管理，方便按材料类型进入不同阅读流。可在设置中定制首页展示的内容来源；删除资料需经确认对话框，成功导入或删除时会有 Toast 与可选音效提示。

## 阅读合集

合集用来把一组相关的阅读材料归在一起，方便按主题、项目或书单集中查看。合集本身不复制内容，只是把阅读库里已有的材料按收录关系组织起来。

### 新建合集

1. 在阅读库点击右上角「+」。
2. 选择「新建合集」。
3. 输入合集名称（例如「系统设计」）。

创建后会直接进入这个空合集。合集名称可以随时在合集卡片菜单中通过「编辑合集名称」修改。

### 往合集里加内容

合集可以收录网页文章、电子书、PDF 和微信读书书刊。有两种方式：

- 在合集内点击「添加到此合集」，从阅读库里勾选已有材料加入。选材弹窗支持按类型筛选和关键词搜索，可以逐个点按「+」加入，也可以直接拖到右侧「待加入」区。
- 在阅读库里把材料卡片直接拖到目标合集卡片上，也能完成加入。

在合集内打开某条材料的卡片菜单，选择「从合集中移除」即可移出；这只会断开收录关系，不会删除原始文章或书刊。

### 合集封面与置顶

合集卡片会用前 9 个成员的封面拼成封面堆叠，并显示成员数量。空合集会显示占位封面。你可以把常用的合集或材料「置顶」，它们会排在阅读库顶部。

### 删除合集

删除合集只移除合集本身和它的收录关系，合集内的文章、电子书、PDF 和微信读书内容会继续留在阅读库。删除前会有确认对话框说明这一点。

## 导入网页文章

1. 在阅读库点击右上角「+」。
2. 选择「添加网页」。
3. 粘贴 `http://` 或 `https://` 开头的文章链接。
4. 等待 Yomitomo 提取标题、作者、正文和图片。

对于需要浏览器渲染的页面，Yomitomo 会尝试使用内置浏览器渲染后再提取正文。默认情况下，本机、局域网和云元数据地址会被拦截；网页 HTML 响应超过 5MB 时也不会导入。

## 导入电子书

1. 在阅读库点击右上角「+」。
2. 选择「电子书文件」。
3. 选择或拖入本地 `.epub`、`.azw3` 或 `.mobi` 文件。

Yomitomo 会解析封面、目录、章节和正文结构。电子书支持批量导入，单次最多 10 本，单本文件大小上限为 80MB。受 DRM 保护或解压后内容过大的文件无法导入。

## 导入 PDF

1. 在阅读库点击右上角「+」。
2. 选择「PDF 文档」。
3. 选择或拖入本地 `.pdf` 文件。

PDF 支持批量导入，单次最多 10 份，单份文件大小上限为 120MB。导入完成后，阅读库会显示页数、文件名和基础元信息。重复导入时，Yomitomo 会提示这份 PDF 已在阅读库，可以直接打开已有文档。

## 筛选、搜索与排序

阅读库支持按阅读状态筛选：

- 全部
- 新添加
- 阅读中
- 已读完

也可以按内容类型筛选，可以多选：合集、网页、电子书、PDF、微信读书。选中的类型会显示成可单独移除的标签，选完所有类型等同于「全部」。这些筛选、搜索词和当前打开的合集会在切换页面或打开文章后保留，返回阅读库时回到原位。

按最近阅读、最近添加、批注数量和讨论数量排序，并用标题或作者关键词搜索。

阅读库卡片会显示材料的阅读进度、划线数量和沉淀数量。PDF 会以文档行展示，网页文章和电子书会保留封面或正文图片。

## 本地保存

阅读数据保存在桌面端应用数据目录。开启「本地保存图片」后，导入网页文章时会把正文图片保存到本机，减少原站图片失效带来的影响。
