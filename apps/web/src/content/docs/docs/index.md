Yomitomo 是本地优先的 AI 伴读桌面应用。它不替你读文章，而是在你阅读时陪你划线、批注、追问和讨论。文章、批注、讨论、Provider 配置和 API Key 都保存在本机。

## 第一次使用

1. 打开 Yomitomo，阅读欢迎介绍后进入应用。
2. 进入「设置」里的「模型与路由」。
3. 添加一个模型供应商，填写 API Key，并测试连接。
4. 为阅读理解助手、深度审阅助手指定任务路由。
5. 回到「阅读库」，导入网页文章、EPUB 电子书或 PDF 文档。

如果要同步微信读书中的划线、想法和阅读进度，先阅读「[获取微信读书 API KEY](/docs/weread-api-key/)」。

## 产品结构

Yomitomo 由几个核心区域组成：

- 「阅读库」管理网页文章、EPUB 电子书和 PDF 文档。
- 「阅读器」负责正文阅读、划线、批注、讨论和沉淀。
- 「助手」管理 AI 阅读助手和审阅助手。
- 「统计」回看本地阅读活动，也可以查询微信读书统计。
- 「设置」配置模型、主题、快捷键、数据管理和更新。
- 「微信读书」通过 API KEY 同步微信读书中的划线、想法和阅读进度。

## 阅读模型

Yomitomo 的核心不是生成摘要，而是保留你的判断链路：

- 划线是原文证据锚点。
- 想法是阅读的最小产出。
- 评论让想法继续被追问和修正。
- AI 助手只在你需要时加入讨论或参与共读。

继续阅读「阅读库与导入」可以完成第一篇材料的导入。
