设置页负责产品边界上的配置：界面语言、外部模型供应商、主题与音效、应用锁、用户输入习惯、本地数据和应用更新。

## 模型供应商

Yomitomo 支持预设供应商和 OpenAI 兼容自定义供应商。配置供应商分为两步：先选择协议服务商，再补充连接信息和模型。

预设供应商包括：

- OpenAI
- Anthropic
- Google Gemini
- DeepSeek
- 阿里云百炼
- Moonshot
- 智谱
- 火山引擎
- 小米 MiMo

第一步选择服务商后，第二步填写名称、Base URL 和 API Key，并配置模型。填写 API Key 后可以点击「获取」从供应商拉取可用模型列表；拉取失败时会回退到预设模型作为候选。预设模型会标注「预设」，自定义模型标注「自定义」。你可以新增、编辑、删除自定义模型，也可以隐藏不需要的预设模型。

API Key 会由系统安全存储：在 macOS 上保存到钥匙串，在 Windows 上保存到凭据管理器。Yomitomo 只在本机保留 provider 配置和对密钥的引用，不保存密钥本身。

## 微信读书

微信读书同步需要单独配置微信读书 API KEY。获取方式见「[获取微信读书 API KEY](/docs/weread-api-key/)」。

配置好 API KEY 后，可以在「同步模式」里选择：

- **手动**：仅在阅读库点击「同步微信读书」时同步。
- **自动**：应用启动后同步一次，之后每 30 分钟在后台自动同步。

自动模式不会和手动同步冲突：如果上一次同步还在进行，会跳过当次定时触发。

## 任务路由

配置供应商后，需要为不同任务指定模型：

| 任务         | 用途                        |
| ------------ | --------------------------- |
| 阅读理解助手 | 划线想法生成和 `@` 提及回复 |
| 深度审阅助手 | 证据、逻辑和表达复核        |

## 语言

在「设置 > 通用」中可切换界面语言（如简体中文与 English）。切换后应用 UI、助手角色说明与部分提示会按语言展示。

## 主题与阅读器纸张

左侧导航中的主题按钮可以切换亮色、暗色、黄昏靛蓝等主题与阅读器纸张；纸张与主题对话框采用手绘墨迹视觉。阅读器纸张只影响网页文章、EPUB 和 PDF 的正文阅读区域。暗色主题下，PDF 会保留原始页面颜色，避免文档内容被反色影响可读性。

## 音效

「设置 > 通用」可开关应用内音效并调节响度，例如导入成功、删除资料、高亮创建、沉淀提交、应用锁解锁与讨论中助手书写时的反馈音。

## 应用锁

「设置 > 通用」可开启应用锁并设置 PIN。开启后，Yomitomo 会在锁定界面要求输入 PIN 才能继续访问本地阅读空间。PIN 校验材料由系统安全存储：在 macOS 上保存到钥匙串，在 Windows 上保存到凭据管理器。

## 快捷键

消息发送可以选择 `Enter` 直接发送，或 `Cmd/Ctrl+Enter` 发送。阅读器选区操作支持自定义复制和批注快捷键，按键必须是单个字母，且不能重复。

## 数据管理

数据管理提供查看数据目录、日志和数据库文件的入口，可设置日志保留时间、清空日志，也支持备份和还原 SQLite 数据库。数据库备份不包含保存在系统钥匙串或凭据管理器中的模型 API Key，也不包含单独保存的电子书原文件。

## 网页导入安全

「设置 > 通用」中可以选择是否允许网页文章导入访问本机、局域网和云元数据地址。默认关闭；只有明确需要导入内网文章时才建议开启。

## 应用更新

Yomitomo 支持 macOS 和 Windows 更新流程。应用启动后会自动检查新版本，之后每 24 小时静默检查一次；发现新版本时不会弹出对话框打断阅读，而是在顶部导航显示「有新版本」徽标，点击即可进入「设置 > 关于」下载和安装。你也可以在「设置 > 关于」中手动「检查更新」。发现新版本时会展示对应版本的更新说明（更新前从官网读取，更新后使用安装包内文案）。macOS 公开安装包经签名与公证。

## 隐私与遥测

「设置 > 通用」中的「隐私」分组提供「发送匿名版本与系统指标」开关，默认开启。开启后，Yomitomo 每天最多发送一次匿名心跳，包含匿名安装 ID、应用版本、平台、系统版本、架构、本地日期和可选时区，用于了解活跃版本与系统分布。它不采集阅读内容、书名、批注、文件路径或 AI 对话。你可以随时关闭这个开关停止上报。

## 助手诊断

如果助手没有按预期回复，可以在设置中打开「助手调用链路」查看最近一次调用状态。这个页面主要用于排查模型配置、网络或供应商返回错误。
