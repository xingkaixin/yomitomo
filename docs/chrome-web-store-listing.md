## 名称

Yomitomo

## 类别

Education

## 简短说明

把网页文章变成可高亮、可批注的阅读器。

## 详细说明

Yomitomo 是一个面向深度阅读的浏览器扩展。用户点击扩展按钮后，可以把当前网页文章转换为干净的阅读视图，在页面中标记重点、写下批注，并围绕批注继续讨论。

扩展可以独立用于当前网页阅读和批注。用户安装 Yomitomo 桌面端并完成配对后，还可以把需要保留的文章和批注保存到本机阅读库。

主要功能：

- 打开当前网页的阅读器视图。
- 选中文本后创建高亮和批注。
- 在批注下继续评论。
- 将文章和阅读记录保存到本机。
- 通过本机连接与 Yomitomo 桌面端同步数据。

Yomitomo 只在用户主动操作时读取当前标签页内容。阅读数据默认保存在用户浏览器和本机环境中。

## 权限理由

### storage

Yomitomo 使用 storage 保存扩展状态、桌面端配对信息、文章记录、阅读状态、高亮、批注和评论数据，用于恢复用户的阅读进度、批注内容和本机同步状态。

### scripting

Yomitomo 使用 scripting 在用户主动点击扩展按钮时向当前标签页注入内容脚本，用于检测文章正文、打开阅读器模式、提取当前页面内容并支持页面内高亮和批注交互。

### activeTab

Yomitomo 使用 activeTab 在用户主动点击扩展按钮后临时访问当前标签页，用于读取当前页面的文章正文、标题和链接，打开阅读器模式，并支持本次页面内的高亮、批注和评论交互。

### host permissions

Yomitomo 请求 127.0.0.1 主机权限，用于连接用户本机运行的 Yomitomo 桌面端。扩展通过本地 HTTP / WebSocket 检测连接状态、发送用户主动保存的文章，并同步高亮、批注和评论数据。

## 远程代码

选择：不，我并未使用远程代码

Yomitomo 扩展包内包含运行所需的 JavaScript 代码。扩展通过本机 HTTP / WebSocket 传输用户数据，所有执行代码随扩展包发布。
